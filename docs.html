Layers

Una layer (capa) es una sección que se pone sobre otra para mostra contenido o secciones diferente de la app

Creacion de una Layer

  • Debe contener la clase layer
  • Debe contener el id de la siguiente forma layer-[nombreDeCapa]
  • Debe contener data-layer-title = "titulo de la capa", este titulo aparecera en el header cuando la capa actual este activa
<div class="layer" id="layer-quiz" data-layer-title="juega"></div>

Mostrar una capa

Para mostrar o abrir una capa hay dos formas, mediante javascript directo y con elementos html

Mostrar capa con js

Para abrir una capa simplemente se llama a la función openLayer(layerName): ejemplo:

openLayer("quiz");

Mostrar capa con html

Para ello debemos asignar los siguientes atributos a cualquier elemento en html:

  • Debe contener la clase trigger-layer
  • Debe contener data-open-layer="nombreDeCapa"

Ejemplo con el elemento button:

<button class="trigger-layer" data-open-layer="quiz">Jugar</button>
